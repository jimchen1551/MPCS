# Implicant
- A **product term** is an implicant of a function. 
e.g., $$F=\Sigma m(0, 1, 4, 5, 9, 13)$$
| ab\cd | 00  | 01  | 11  | 10  |
| :---: | :-: | :-: | :-: | :-: |
| 00    | 1   | 1   | 0   | 0   |
| 01    | 1   | 1   | 0   | 0   |
| 11    | 0   | 1   | 0   | 0   |
| 10    | 0   | 1   | 0   | 0   |
- Implicants existed: 
  a'b'c'd', a'b'c'd, a'bc'd', a'bc'd, abc'd, ab'c'd, 
  a'b'c', a'bc', a'c'd', a'c'd, bc'd, ac'd, 
  a'c', c'd
