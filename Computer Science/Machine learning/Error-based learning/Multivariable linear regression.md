# Multivariable linear regression
$$\mathbf{M}_w(d)=
\begin{bmatrix}
w[0]&w[1]&\dots&w[m]
\end{bmatrix}\cdot
\begin{bmatrix}
1\\d[1]\\\vdots\\d[m]
\end{bmatrix}=w^Td$$
$$L_2(\mathbf{M}_w, D)=\frac{1}{2}\|t-\mathbf{M}_w(d)\|_2^2=\frac{1}{2}\|t-w^Td\|_2^2$$
