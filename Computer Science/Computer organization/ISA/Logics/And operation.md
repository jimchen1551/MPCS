# And operation
| Instruction | Comment |
| ----------- | ------- |
| and         |         |
| andi        |         |
```MIPS
and $t0, $t1, $t2  % $t0 = $t1 & $t2
```
- useful to mask bits in a word (selecting some bits and clearing others to 0)
