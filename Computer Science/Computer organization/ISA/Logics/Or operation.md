# Or operation
| Instruction | Comment |
| ----------- | ------- |
| or          |         |
| ori         |         |
| xor         |         |
| xori        |         |

```MIPS
or  $t0, $t1, $t2  % $t0 = $t1 | $t2
```
- useful to include bits in a word (setting some bits to 1 and leaving others unchanged)
